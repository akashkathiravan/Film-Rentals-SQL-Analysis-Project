## Film-Rentals-SQL-Analysis-Project
This project contains a set of SQL queries and exploratory analyses performed on a Film Rental database. The queries focus on understanding customer behavior, rental trends, revenue patterns, and store performance.

## ER Diagram

![ER Diagram](./Screenshot%202025-11-22%20131409.png)

This repository includes a full SQL project file containing all analytical queries, data checks, and transformations used in this Film Rental analysis.

MySQL
SQL querying (DDL, DML, DQL)
Relational database design
ER modeling
